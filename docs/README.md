# ARTH.OS Documentation Hub

This directory is the product source of truth for ARTH.OS.

## Purpose

These documents define the enterprise product blueprint, architecture, and module standards so ARTH.OS is built as a platform rather than a collection of isolated features.

## Documentation Structure

- [00-product-vision.md](00-product-vision.md) – product vision, market focus, and platform principles
- [01-architecture/README.md](01-architecture/README.md) – enterprise architecture and engineering standards
- [02-enterprise-design-system/README.md](02-enterprise-design-system/README.md) – EDS v1.0 foundation
- [03-eom/README.md](03-eom/README.md) – enterprise organization management blueprint
- [04-iam/README.md](04-iam/README.md) – identity, access, and security blueprint
- [05-hrms/README.md](05-hrms/README.md) – HRMS platform blueprint

## Module Standard

Every future module should include:

1. Business overview
2. Functional requirements
3. User stories
4. UX and navigation
5. Database design
6. API contracts
7. Workflow definitions
8. Permissions and audit
9. Reports and analytics
10. Test strategy
