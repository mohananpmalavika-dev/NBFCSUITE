Welcome to the NBFC Financial Suite! Follow these simple steps to get started.

================================================================================
  OPTION 1: AUTOMATED STARTUP (RECOMMENDED)
================================================================================

Just double-click one of these files:

  📄 start-dev.bat         - Automatically sets up and runs everything
  
This will:
  ✓ Check prerequisites (Python, Node.js)
  ✓ Create virtual environment (if needed)
  ✓ Install all dependencies
  ✓ Set up configuration files
  ✓ Run database migrations
  ✓ Start backend server (Port 8000)
  ✓ Start frontend server (Port 3000)
  ✓ Open application in browser

Wait 10-20 seconds after starting for servers to be ready.

================================================================================
  OPTION 2: MANUAL STARTUP
================================================================================

1. SETUP DATABASE (First time only)
   ---------------------------------
   • Make sure PostgreSQL is installed and running
   • Run: .\setup-database.ps1
   • Follow the prompts to create database

2. START BACKEND
   --------------
   • Open PowerShell in project root
   • Run:
     cd backend
     .\venv\Scripts\activate
     python -m uvicorn backend.main:app --reload

3. START FRONTEND (New PowerShell window)
   ----------------------------------------
   • Open new PowerShell in project root
   • Run:
     cd frontend\apps\admin-portal
     npm run dev

================================================================================
  USEFUL COMMANDS

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Python not found"
Solution: Install Python 3.11+ and add to PATH

Problem: "Node not found"
Solution: Install Node.js 20+ and add to PATH

Problem: "Database connection failed"
Solution: 
  • Make sure PostgreSQL is running
  • Check credentials in backend\.env file
  • Run .\setup-database.ps1

Problem: "Port already in use"
Solution: 
  • Run .\stop-dev.ps1 to stop existing servers
  • Or manually kill processes on ports 3000 and 8000

Problem: "Module not found"
Solution:
  • Backend: cd backend && .\venv\Scripts\pip install -r requirements.txt
  • Frontend: cd frontend\apps\admin-portal && npm install --legacy-peer-deps
