================================================================================
  🚀 QUICK START (RAILWAY - RECOMMENDED)
================================================================================

If you want to deploy RIGHT NOW, run these commands:

1. Install Railway CLI:
   npm install -g @railway/cli

2. Login:
   railway login

3. Deploy Backend:
   cd c:\NBFCSUITE\backend
   railway init
   railway add      (choose PostgreSQL)
   railway up

4. Set Variables:
   railway variables set SECRET_KEY=your-key-here
   railway variables set JWT_SECRET_KEY=your-jwt-key-here

5. Run Migrations:
   railway run alembic upgrade head

6. Deploy Frontend:
   cd ..\frontend\apps\admin-portal
   railway init     (link to existing project)
   railway variables set NEXT_PUBLIC_API_URL=your-backend-url/api/v1
   railway up

7. Get URLs:
   railway domain

DONE! Your app is live! 🎉
