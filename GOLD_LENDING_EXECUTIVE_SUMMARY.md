## 🏗️ Technical Architecture

### Technology Stack
```
Backend:  Python 3.11 + FastAPI + SQLAlchemy
Frontend: Next.js 14 + TypeScript + Tailwind CSS
Database: PostgreSQL with JSONB
API:      RESTful with OpenAPI/Swagger
```

### Key Design Decisions
1. **Microservices Architecture**: Gold service is independent
2. **API-First Design**: All operations via REST APIs
3. **Database-Driven Config**: Business rules in database
4. **Type Safety**: TypeScript + Pydantic validation
5. **Audit-First**: All operations logged

### Scalability Features
- Async/await throughout
- Database connection pooling
- Indexed queries
- Stateless API design
- Horizontal scaling ready
