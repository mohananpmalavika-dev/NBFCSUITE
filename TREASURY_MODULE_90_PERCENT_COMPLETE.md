## 🚀 Deployment Information

### System Requirements
- Python 3.11+
- PostgreSQL 15+
- Node.js 18+
- NPM/Yarn

### Deployment Steps

#### 1. Database Setup
```bash
cd backend
pip install -r requirements.txt
alembic upgrade head
```

#### 2. Start Backend
```bash
cd backend
python main.py
```
**Access:** http://localhost:8000  
**API Docs:** http://localhost:8000/docs

#### 3. Start Frontend
```bash
cd frontend/apps/admin-portal
npm install
npm run dev
```
**Access:** http://localhost:3000/treasury

### Production Build
```bash
# Backend
gunicorn -w 4 -k uvicorn.workers.UvicornWorker main:app

# Frontend
npm run build
npm start
```
