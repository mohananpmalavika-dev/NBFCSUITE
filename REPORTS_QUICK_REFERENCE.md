## 🚀 Common Actions

### View Dashboard
1. Navigate to **Reports & Analytics**
2. Dashboard loads automatically
3. Review 7 KPI cards
4. Check trend charts
5. Auto-refreshes every 60 seconds

### Generate Report
1. Click **"Reports"** tab
2. Browse available reports
3. Click **"Generate"** on desired report
4. System shows Report Viewer
5. View summary and details

### Apply Filters
1. In **Report Viewer** tab
2. Select **Period** dropdown
3. Choose **Branch** (optional)
4. Set **Date Range** (optional)
5. Click **"Apply Filters"**

### Export Report
1. Click **"Export"** button (top right)
2. Select **Report Type**
3. Choose **Export Format**
4. Click **"Export"**
5. File downloads automatically

## 🎯 Dashboard Metrics Explained

### Total Lockers
- **What**: Inventory count
- **Breakdown**: Small, Medium, Large, Extra Large
- **Use**: Capacity planning

### Occupancy Rate
- **What**: % of lockers in use
- **Calculation**: (Occupied / Total) × 100
- **Benchmark**: 70-80% is healthy

### Rent Collection
- **What**: Current month collections
- **Metrics**: Expected, Collected, Pending, %
- **Benchmark**: 85%+ is good

### Overdue Rent
- **What**: Unpaid rent count
- **Metrics**: Count, Total amount
- **Action**: Follow up on 30+ days

### Waiting List
- **What**: Customers waiting
- **Metrics**: Total count, Average wait days
- **Action**: Allocate when available

### Recent Activity
- **Allocations**: New customers
- **Surrenders**: Closed lockers
- **Trend**: Compare daily/weekly/monthly

## 🔢 Key Formulas

### Occupancy Rate
```
Occupancy % = (Occupied Lockers / Total Lockers) × 100
```

### Collection Percentage
```
Collection % = (Amount Collected / Amount Expected) × 100
```

### Growth Rate
```
Growth % = ((Current - Previous) / Previous) × 100
```

### Average Wait Time
```
Avg Wait = Sum of (Today - Added Date) / Total Waiting
```
